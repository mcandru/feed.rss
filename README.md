# feed.rss

A minimal RSS feed reader that stores your news feed in local storage.

Inspired by [this tweet](https://x.com/karpathy/status/2018043254986703167) from Andrej Karpathy, the goal of the feed reader is to help curate more long-form reading and cut out the AI slop.

## Tech Stack

- Vue.js
- TypeScript
- tailwind CSS
