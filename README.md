# feed.rss

A minimal RSS feed reader that stores your news feed in local storage.

Inspired by this tweet from Andrej Karpathy, the goal of the feed reader is to help curate more long-form reading and cut out the AI slop.

## Tech Stack

- Vue.js
- TypeScript
- tailwind CSS
- Vercel for deployment
- Vercel serverless function for CORS proxy for RSS feeds
